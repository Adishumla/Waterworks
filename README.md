## Getting Started
First, install the project:
```bash
npm install
```

To develop, run the development server:
```bash
npm run dev
```

To build and run a build version:
```bash
npm run build
npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
